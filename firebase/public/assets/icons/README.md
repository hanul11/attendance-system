# LOGIFLOW App Icon Assets

실제 아이콘 이미지는 이번 Sprint에서 생성하지 않습니다. 아래 파일명을 그대로 사용하면 `manifest.webmanifest`, Android Adaptive Icon, iPhone App Icon 구조와 연결됩니다.

| 파일 | 크기 | 용도 |
| --- | --- | --- |
| `icon-192.png` | 192 x 192 | Android/PWA 기본 아이콘 |
| `icon-512.png` | 512 x 512 | Android/PWA 고해상도 아이콘 |
| `icon-maskable-512.png` | 512 x 512 | Android Adaptive Icon 전경 안전영역 포함 |
| `apple-touch-icon-180.png` | 180 x 180 | iPhone 홈 화면 아이콘 |

모든 파일은 PNG, sRGB, 투명 배경 또는 브랜드 배경으로 제작합니다. Maskable Icon의 핵심 로고는 중앙 80% 안전영역 안에 배치합니다.
