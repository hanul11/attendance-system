# 한울 출퇴근 기록 PWA 아이콘

회사 CI 원본 `outputs/hanul-company-logo-transparent.png`에서 꽃 심볼만 사용한 설치 아이콘입니다. 심볼의 형태와 색상은 변경하지 않았습니다.

| 파일 | 크기 | 용도 |
| --- | --- | --- |
| `icon-192.png` | 192 x 192 | Android/PWA 기본 아이콘 |
| `icon-512.png` | 512 x 512 | Android/PWA 고해상도 아이콘 |
| `icon-maskable-512.png` | 512 x 512 | Android Adaptive Icon용 안전 여백 포함 아이콘 |
| `apple-touch-icon-180.png` | 180 x 180 | iPhone 홈 화면 아이콘 |

모든 파일은 PNG 형식이며 배경색은 앱 배경과 동일한 `#F7F8FA`입니다. 일반 아이콘은 꽃 심볼을 캔버스의 약 72% 크기로, maskable 아이콘은 잘림 방지를 위해 약 66% 크기로 중앙 배치했습니다.
